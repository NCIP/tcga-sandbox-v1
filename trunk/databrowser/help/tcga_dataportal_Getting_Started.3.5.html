Step 2. Apply Sample Thresholds
Once the value threshold is applied, a sample threshold completes the process to create a final table of results listing the genes and number of samples residing in each anomaly category. There is a sample threshold percentage value for each Anomaly category (Mutation, Overexpression, Underexpression, Amplification, Deletion). The following table shows a list of example sample thresholds. These are examples and may not be the the current values used in TCGA Data Portal Prototype. As you can see, the lower the sample threshold, the more anomalies appear in the list.
Because the same genes appear in multiple pathways, you get more pathways than genes for any threshold level.
