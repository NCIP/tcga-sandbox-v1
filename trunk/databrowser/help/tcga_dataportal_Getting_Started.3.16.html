Ambiguous Probes
Some Affymetrix probes are non-specific, that is, they may hybridize to several homologous transcripts. In this case, several Entrez IDs are reported for a single probe. This creates ambiguous cases where the same gene has multiple expression values. For example, Probe 1 maps to genes A and B. Probe 2 maps to genes B and C. If Probe 1 and Probe 2 report different expression levels, then creating a single summary value for gene B is difficult. In these cases, we have eliminated the gene from our output set.
