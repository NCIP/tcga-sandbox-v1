Step 1. Calculate Amplified and Deleted Regions
To find the amplified and deleted regions, including gene and non-gene regions, DNAcopy was used which is part of the Bioconductor package. All sequence data for each chromosome was processed. The Value Threshold was applied to the raw copy number data to determine Amplified and Deleted genes. The following table lists the Anomaly categories and the associated Value Threshold applied.
Generally, a gene has to be completely contained within an amplified or deleted region to be an anomaly. Currently in the TCGA Data Portal Prototype protype, only a portion of the gene has to overlap a region to be considered an anomaly.
