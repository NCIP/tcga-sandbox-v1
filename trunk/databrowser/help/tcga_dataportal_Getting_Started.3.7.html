Mutation Processing
The mutation data was processed as follows:
1.
Determine the Entrez IDs for each gene in BCM via the
CGAP.ll_gene CGAP.ll_alias tables.
•
Downloaded the entire table’s Entrez IDs and their HUGO symbols.
•
Performed a LINUX join command on the HUGO symbol.
2.
Limited to BCM genes that are present in any of the BioCarta pathways.
•
Queried the TCGA Data Portal Summary database.
3.
Limited to BCM genes that have a mutation (4th column in BCM file >= 1).
4.
Inserted this final BCM dataset into the TCGA Data Portal Summary database.
5.
Determined any additional genes found in WIBR.
•
Performed a LINUX join command between the original two data files using the HUGO symbol column.
6.
Limited to WIBR genes that are present in any of the BioCarta pathways.
•
Query the TCGA Data Portal Summary dabase.
7.
Limited to WIBR genes that have a mutation (4th column in BCM file >= 1).
8.
Inserted this final WIBR dataset into the TCGA Portal Summary database.
