Determining Agent Associations
The agent data was extracted from the findings of the Cancer Gene Curation Project. This project was a literature-mining exercise to find the following associations:
•
Associations between genes and types of cancer
•
Associations between genes and agents (drugs)
In the first step of this literature-mining project, candidate assertions in abstracts of publications in PubMed were identified. In the second step, a human curator examined and recorded whether these candidate assertions were actual assertions.
For detailed information about this project, see http://ncicb.nci.nih.gov/NCICB/projects/cgdcp.
The TCGA Data Portal Prototype includes in the final results only associations for which the human curator thought it was an assertion, as well as a positive assertion. For example, TCGA Data Portal Prototype would not include an assertion, such as, “activity of AKT1 failed to show any correlation with the level of Gleevec,” because it is not a positive assertion.
