Installing the SVG Plug-in
For the TCGA Data Portal Prototype to display the pathway diagrams, you must install the Adobe SVG Plug-in (see http://www.adobe.com/svg/viewer/install/main.html). If you are an Internet Explorer user, Adobeâ€™s installer will add the plug-in automatically into Internet Explorer.
If you are a Mozilla Firefox user, follow these steps:
9.
Download and install a beta version of the Adobe plug-in. Unfortunately, Firefox is not fully compatible with the current version of the plug-in.
10.
Using Windows Explorer browse to the
C:\Program Files\Common Files\Adobe folder.
Here you will find an SVG Viewer folder and inside it a plugins folder.
Inside the Plugins folder you will find two files: NPSVG6.dll and NPSVG6.zip.
11.
Copy both files into your Firefox plugins folder (which is likely to be
C:\Program Files\Mozilla Firefox\plugins).
12.
In the Firefox location bar, type: about:config
13.
Search for SVG.
14.
Set svg.enabled to false (you can do this by double clicking on the row)
15.
Close all browser windows and restart Firefox.
