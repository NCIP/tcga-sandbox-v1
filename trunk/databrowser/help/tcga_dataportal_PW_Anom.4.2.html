Viewing Pathways and Associated Abnormalities
The data on this site is preliminary and intended only to demonstrate the functions of this Alpha release. Use the data at your own risk. This data is subject to change. We welcome your feedback on this release and its associated data. Please email us at ncicb@pop.nci.nih.gov with questions or comments.
The Pathways and Associated Abnormalities page lists the pathways alphabetically and indicates with a black dot whether the pathway has certain gene anomalies associated with it. For detailed information about how TCGA Data Portal Prototype determines the abnormality type and associated drug agents, see About TCGA Data Portal Prototype.
The following table describes each column on the Pathways and Associated Anomalies table.
Column Name
Description
Any Anomaly/Agent
A black dot appears when genes within the pathway has an anomaly and/or drug agent associated with it.
Mutated
A black dot appears when genes within the pathway are categorized as mutated within TCGA Data Portal Prototype. For more information on categorizing mutations, see Determining Mutations on page 3.
Amplified
A black dot appears when genes within the pathway are categorized as amplified within TCGA Data Portal Prototype. For more information on categorizing copy number data, see Determining Anomalies for Copy Number on page 2.
Deleted
A black dot appears when genes within the pathway are categorized as deleted within TCGA Data Portal Prototype. For more information on categorizing copy number data, see Determining Anomalies for Copy Number on page 2.
Agents
A black dot appears when genes within the pathway are associated with a drug agent TCGA Data Portal Prototype. For more information on determining agent association, see Determining Agent Associations on page 5.
