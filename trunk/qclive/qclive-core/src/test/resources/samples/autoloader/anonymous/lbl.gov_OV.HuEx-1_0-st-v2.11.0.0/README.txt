The data archive contains the Cancer Genome Atlas (TCGA) gene expression analysis using Affymetrix Exon 1.0 arrays of the Oct 2008 Ovary Serous Cystadenocarcinoma RNA Samples (batch 11) performed by the LBNL Characterization Center.

This archive contains data for microarray assays on 37 BCR test tumor samples of the Oct 2008 ovarian, and 3 control samples. The data for each assay is organized in MAGE-TAB format. In addition the CEL file for each scan is provided.
