Building the XML Document

To build the XML data document for this data, again launch our XML Generator Utility and choose option 1 "I want to manually enter data".

Now, first enter the data series name in the grid (2005, 2006 etc.) by clicking on column header and changing it from Series_1 (or Series_2 ...) to our required value (2005, 2006). Thereafter enter the actual values as shown below:
