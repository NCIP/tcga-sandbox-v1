Overview

FusionCharts XT helps you create interactive charts for web and enterprise applications. It is the industry's leading enterprise-grade charting component that functions seamlessly on PCs, Macs, iPads, iPhones and a majority of other mobile devices. The stunning looks, refreshing animation and power-packed reporting capabilities of FusionCharts have enabled our 19,000 customers and 400,000 users to add "wow" to their applications.

FusionCharts leverages JavaScript (HTML5) and Flash to create stunning data visualizations unmatched by conventional server-side components. As it works with both XML and JSON data, it can be integrated with any server-side technology (ASP, ASP.NET, PHP, JSP, ColdFusion and Ruby on Rails etc.) and database.

Be it a simple column or pie chart, a combination chart, advanced zoom & scroll charts or specialized sales & marketing charts, FusionCharts covers all your charting requirements. All the charts support interactive options like tooltips, drill-down, exporting as image/PDF/CSV and JavaScript integration.

Advantages of using FusionCharts
Works with all web technologies and databases
 

FusionCharts works with ASP, ASP.NET, PHP, JSP, ColdFusion, Ruby on Rails, JavaScript or even simple HTML pages. It connects to any database (MS SQL, MySQL, Oracle and PostgreSQL) and web service to enable real-time charting.

 Stunning looks & complete reporting
 

FusionCharts not only gives your data a pretty face, it also adds smart reporting features like tooltips, drill-down, chart export, selective display of data and scroll & zoom to your chart. Some charts also allow visual editing of data using simple drag-n-drop.
Animated and interactive Charts that "wow" your audience
 

Using FusionCharts, you can quickly and easily render animated charts that have a lot of interactive options for the end users. Different chart types support different forms of animation and interactivity, thereby providing a unique experience to the end users.
Accepts both XML and JSON data
 

FusionCharts accepts both XML and JSON data, enabling you to use a data format of your choice. Using XML you can connect it to your enterprise applications very easily and with JSON to your JavaScript frameworks.
Reduces load on your servers
 

In traditional image based charting systems, the charts are generated as image at the server side. So, for each chart that you need to serve to a user, you'll have to build complex images on the server and then stream it to the client. When the need of hour is high, this can be an expensive resource on the server, as image creation takes a toll on the server.

At this juncture, FusionCharts brings you great relief - all the charts are rendered at client side using the widely installed Adobe Flash Platform. The server is just responsible for streaming the pre-built SWF/JS files and your XML/JSON data files to the end viewers. From there on, Flash Player takes the onus of rendering the charts. Also, the chart SWF/JS files can be cached so that you can just update the data, and not send chart SWF files every time.
JavaScript (HTML5) and Flash based charting
 

FusionCharts renders charts using JavaScript (HTML5) and Flash, giving you the best of both worlds. As such, it is the leading enterprise-grade charting component that works seamlessly on PCs, Macs, iPads, iPhones and most other mobile devices.
Wide variety of charts supported
 

Apart from the standard charts like pie, bar, line and area, FusionCharts also supports advanced chart types like combination, scroll, zoom line, XY Plot, Marimekko and Pareto charts. The charts support advanced options like dual axis, trend lines/zones, missing data, advanced number formatting, color themes and customizable tooltips.
AJAX-enabled charts with extensive JavaScript API
 

All the charts in FusionCharts Suite are tightly integrated with JavaScript. This allows client-side chart or attribute update, listening to chart events, maintaining chart states or cloning it.
Unlimited levels of drill-down supported
 

FusionCharts is a perfect fit for reports with multiple levels of drill-down. Individual data points on the chart can be configured to drill-down in same window, new window, pop-up window or frames. FusionCharts also introduces LinkedCharts, an industry first, which lets you create unlimited drill down levels from a single data source; that too, without any additional code.
Exports to JPG, PNG, PDF and CSV
 

While charts are generated using Flash, they can also be exported to JPG, PNG or PDF. The data can also be copied from the chart as CSV. This can be done either through user interaction or even programmatically.
Can handle high data load in zoom chart along with innovative comparison methods
 

The newly introduced zoom chart allows you to plot thousands of point in the chart with no clutter and high level of usability. This is possible using the interactive zoom and scroll functionalities present in the chart. Additionally, the pin method allows a unique and interactive way for users to compare a subset of data against another subset.
Industry-leading component trusted by Fortune 500 companies and the web’s most popular sites
 

With over 19,000 customers and 400,000 users, FusionCharts is the industry's leading RIA charting component. A majority of Fortune 500 companies prefer FusionCharts for their reporting needs. Popular websites that trust FusionCharts for their data visualization needs include LinkedIn, Weather.com, Federal IT Dashboard, AWeber, PollDaddy, StatCounter and Wufoo.

Commercially open source
 

You can purchase the entire source of FusionCharts as a part of the Enterprise, Enterprise Plus, SaaS or OEM License.  This allows you to modify and recompile it as per your requirements. The code is written in ActionScript 2 and can be compiled using Flash IDE (8/CS3/CS4/CS5).
