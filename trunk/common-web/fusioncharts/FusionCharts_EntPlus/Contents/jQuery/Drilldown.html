Configuring LinkedCharts using FusionCharts jQuery plugin

FusionCharts jQuery also provides option to configure LinkedCharts. You would need to use drillDownFusionChartsTo method to achieve this.

To aid your understanding of this section, we would recommend you to go through the following sections of documentation (if you've not already read them):
  • FusionCharts and JavaScript > Using LinkedCharts
  • FusionCharts and JavaScript > Using LinkedCharts > Configure
  • FusionCharts and JavaScript > API > Functions > ConfigureLink
  • FusionCharts & jQuery > Overview

The drillDownFusionChartsTo method is jQuery adoption of configureLink() function provided by FusionCharts JavaScript class. You can use this method to configure all the child charts created using LinkedCharts feature.

The code snippet below configures the first-level child charts (of all charts present in the HTML elements matching id - "chartContainer") to render as Pie 3D chart and show a "close" button at top:

$("#chartContainer").drillDownFusionChartsTo
    ({ swfUrl : "Charts/Pie3D.swf", overlayButton: { message: 'close', bgColor:'FFEEEE' } });

See it live!

The code snippet below configures the first two levels of child charts (of all charts present in the HTML elements matching id - "chartContainer") to render as Pie 3D and Line charts, respectively:

$("#chartContainer")
    .drillDownFusionChartsTo({ swfUrl : "Charts/Pie3D.swf" }, {  swfUrl : "Charts/Line.swf" } );

The above setting can also be passed as :

$("#chartContainer")
    .drillDownFusionChartsTo({ swfUrl : "Charts/Pie3D.swf" })
          .drillDownFusionChartsTo({  swfUrl : "Charts/Line.swf" });

The code below shows how settings for all levels can also be passed as an array:

$("#chartContainer").drillDownFusionChartsTo
     (
        [
          { overlayButton: { message: ' « Annual Sales'   } },
          { overlayButton: { message: ' « Quarterly Sales'} },
          {
            swfUrl: 'Charts/Pie3D.swf',
            width: '225', height: '200',
            renderAt: 'weeklyChartContainer',
            insertMode: 'append',
            overlayButton: { message: 'Close' }
          }
        ]
      );

Please note that passing of configuration objects as a single array or as a series of parameters doesn't make a difference to the order or the level, to which they get assigned.
