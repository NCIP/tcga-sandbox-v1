Welcome to the TCGA DCC Prototypes page
The following pages are available for exploration.
  • Bunsen - Prototype for an interactive selector of clinical data
    This is a proposal only so far, which is why it has a totally unofficial name. ;-) Bunsen pulls in a whole set of clinical parameters to the front end and lets the user play with the parameters while the number of patients within the selected values is continually adjusted. The adjustment takes place both for the number of patients in the selected parameter and from the interaction with all of the parameters. Hmm, I explained that poorly. Just play with it and you'll see what I mean.
  • Watson - Prototype for the new version of the Data Browser
    This is only partially complete. Pick a disease and click on "Define Patient Classes". No results yet and just showing patient classes. The button for go on the 2nd page does not yet work. More updates as I go along.

  • Directory Access Interface
    Two potential version of a javascript intermediated directory access interface to the TCGA data. Note: Only the GBM directory tree, and the first directory of each of it's child directories, is populated. In both cases, this is a very limited implementation. There are lots of behavioral tweaks I would add in putting this out for public consumption.
    • DAI Version 1
      A version with a single tree window where you just walk down the directory tree in that window. Double cliking on a directory in the panel will open the directory, or you can use the +/- toggles.
    • DAI Version 2
      A version with two windows, one allows you to walk down the directory tree. The other window just shows the files in a single directory. The synchronize button, when pressed, will synchronize the two windows. Double cliking on a directory in either panel will open the directory.
