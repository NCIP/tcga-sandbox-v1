Data Level Level Type Description Example
1 Raw Low-level data for a single sample, not normalized across samples, and not interpreted for the presence or absence of specific molecular abnormalities. Sequence trace file; Affymetrix CEL file
2 Processed Data for a single sample that has been normalized and interpreted for the presence or absence of specific molecular abnormalities. Mutation call for a single sample; amplification/deletion/LOH call for a probed locus in a sample; expression of a splice variant.
3 Segmented/ Interpreted For genomic copy-number assays, segmented data is processed data for a single sample that has been further analyzed to aggregate individual loci into larger contiguous regions. Amplification/deletion/LOH call for a region in a sample.
4 Summary Finding A quantified association, across classes of samples, among two or more specific molecular abnormalities, sample characteristics, or clinical variables. A finding that a particular genomic region (a "region of interest") is amplified in 10% of TCGA glioma samples.
