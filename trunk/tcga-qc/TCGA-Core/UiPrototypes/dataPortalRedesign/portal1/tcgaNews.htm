08/18/08 - Broad MAF File Correction
Broad had a problem with their last MAF file generation that caused the alleles in the file to be copied in incorrectly. The status descriptions were accurate (validation status, somatic status) but the alleles listed for sequencing & validation genotypes were not. Broad uploaded a new MAF on Friday 8/15, so please use this new file or a newer file for any downstream analysis.
07/15/08 - Error within Clinical Data XML
Normal Tissue type '11' was defined as a tumor in three XML fields and has been corrected. If you used the name of sample/aliquot barcode to determine tumor vs normal, this error does not affect you. If you used the XML, please click here.

04/14/08 - Search by File Name
The search parameter should match a file name and NOT an archive name. You may enter the full file name (e.g. TCGA-02-0074-01A-01D_Segment.tsv) or a part of the name (e.g. TCGA-02-0074). The results list the archives that contain those files. To view the actual files contained in those archives click on "View Files" for each archive.
