TCGA Data Portal Overview
TCGA Data Portal provides a platform for researchers to search, download, and analyze data sets generated by TCGA. It contains clinical information, genomic characterization data, and high-throughput sequencing analysis of the tumor genomes.
We offer several ways to view, access and download the data (these options are available at the top of every page.
