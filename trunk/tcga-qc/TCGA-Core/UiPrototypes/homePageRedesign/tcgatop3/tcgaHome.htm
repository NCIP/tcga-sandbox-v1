04/14/2010 - DCC Web Service
The DCC database is now accessible via a RESTful web service. For information on using this web service, please see the DCC Web Service User Guide.
11/10/2009 - TCGA Data Service on caGrid
A TCGA caGrid data service that allows access to public TCGA data is now available. For more information, please see the TCGA Grid User Guide.
8/5/2009 - TCGA Ovarian Data Freeze 2
A list of the archives that comprise the 08/05/09 Ovarian data freeze is available here. The freeze includes all characterization data through Batch 15, some sequence-trace data through Batch 13. Mutation files included in this freeze can be found here.
06/23/2009 - Clinical Data Available
Updated clinical data for GBM and Ovarian samples are available via the Data Access Matrix: Ovarian Clinical Data, GBM Clinical Data. Clinical archives for both disease types are available via SFTP (caftps.nci.nih.gov).
06/19/2009 - TCGA Sequencing Data
The TCGA network has selected more than 6,000 gene and miRNA targets for sequencing that represent both protein-coding genes and microRNAs (miRNAs). Click here for the TCGA Sequencing Data page.
