04/14/08 - Search by File Name
The search parameter should match a file name and NOT an archive name. You may enter the full file name (e.g. TCGA-02-0074-01A-01D_Segment.tsv) or a part of the name (e.g. TCGA-02-0074). The results list the archives that contain those files. To view the actual files contained in those archives click on "View Files" for each archive.
